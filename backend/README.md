# FlowBoard — Smart Task Manager

Full-stack task management app built with Django + React.

## Tech Stack
- **Backend:** Django, DRF, Django Channels, Celery, PostgreSQL, Redis
- **Frontend:** React, Vite, TailwindCSS (Week 2)
- **DevOps:** Docker, docker-compose

## DSA Applied
- Min-Heap — priority queue for task scheduling
- Binary Search — deadline range filtering
- Stack — undo last status change
- Sliding Window — 7-day analytics

## Features
- JWT authentication
- Real-time WebSocket notifications
- Task CRUD with priority scoring
- Analytics dashboard
- Fully Dockerized

## Run locally
```bash
docker-compose up --build
```
